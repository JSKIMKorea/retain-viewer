# 수동 기사 주입 설정
# ============================================================
# 형식: 카테고리 | 순서 | 기사URL
#
# 카테고리 (정확히 아래 이름 중 하나):
#   - 한국공인회계사회 기사
#   - 업계동향
#   - 회계
#   - 세무
#   - 상법개정
#
# 순서: 해당 카테고리에서 몇 번째에 표시할지 (1 = 맨 위)
#
# 사용 예시 (아래 줄의 # 제거 후 URL 교체):
# 업계동향 | 1 | https://news.example.com/article/12345
# 회계 | 3 | https://news.example.com/article/67890
#
# 주의사항:
#   - 줄 앞에 #이 있으면 무시됩니다 (주석)
#   - 빈 줄도 무시됩니다
#   - news_update.py 실행 시 자동으로 읽어서 처리합니다
#   - 처리 후 이 파일을 비워두면 다음 실행 시 수동 기사 없이 동작합니다
